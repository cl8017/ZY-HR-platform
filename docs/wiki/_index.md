## 🎯 项目目标

建设镇江烟草"人才培养"数智平台，涵盖：

1. **数据看板** — 红色预警、退休预测、编制分析等可视化驾驶舱
2. **人才库** — 人才信息管理、品牌人才、标签体系、数据统计
3. **导师帮带** — 导师关系管理、帮带看板、词云分析
4. **大师工作室** — 工作室创建与管理、成果展示
5. **课题项目组** — 项目生命周期管理（CRUD + 成员/阶段/成果）
6. **文档查重** — docx/pdf 智能查重分析
7. **岗位胜任力** — 员工胜任力雷达图分析
